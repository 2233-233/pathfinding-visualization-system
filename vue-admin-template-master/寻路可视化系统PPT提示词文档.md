# 寻路可视化系统 - PPT生成提示词文档

## 系统概述
这是一个基于Vue.js的寻路算法可视化教学平台，用于演示和比较不同寻路算法的执行过程、性能差异和路径优化效果。

## 一、技术亮点

### 1. 前端技术栈
- **Vue.js 3** - 现代化的前端框架，提供响应式数据绑定和组件化开发
- **Element Plus** - UI组件库，提供美观的界面组件和交互体验
- **SCSS/SASS** - CSS预处理器，支持变量、嵌套、混合等高级特性
- **Axios** - HTTP客户端，处理RESTful API请求和响应拦截
- **Vue Router** - 前端路由管理，支持SPA单页应用
- **Vuex/Pinia** - 状态管理，集中管理应用状态

### 2. 算法实现技术
- **面向对象设计** - 算法基类+具体算法子类的继承体系
- **模块化架构** - 算法、网格、实验数据分离，高内聚低耦合
- **性能优化** - 批量更新、延迟计算、动画帧率控制
- **可扩展性** - 易于添加新的寻路算法和功能模块

### 3. 数据管理
- **实验数据持久化** - 算法执行结果自动保存到后端数据库
- **实时统计** - 运行时间、访问节点数、路径长度等实时计算
- **步骤记录** - 算法执行的每一步都记录并可回放
- **用户实验历史** - 支持查看和对比历史实验记录

### 4. 动画与可视化
- **逐帧动画** - 算法搜索过程的可视化展示
- **速度控制** - 10级动画速度调节（极慢到最快）
- **颜色编码** - 不同状态单元格使用不同颜色标识
- **实时反馈** - 鼠标悬停、点击交互的即时视觉反馈

## 二、功能亮点

### 1. 算法多样性
- **广度优先搜索 (BFS)** - 逐层扩展，保证找到最短路径
- **深度优先搜索 (DFS)** - 深度优先探索，可能找到非最优解
- **Dijkstra算法** - 带权图最短路径，考虑边权重
- **A*算法** - 启发式搜索，结合实际距离和预估距离

### 2. 交互功能
- **地图编辑** - 支持设置起点、终点、障碍物
- **实时修改** - 算法运行中可暂停、继续、重置
- **网格配置** - 多种网格尺寸选择（16×16, 32×42, 42×57）
- **随机生成** - 一键生成随机障碍物布局

### 3. 可视化控制
- **搜索过程显示** - 可控制是否显示算法搜索过程
- **最短路径显示** - 可控制是否显示最终找到的路径
- **网格线显示** - 可控制是否显示网格辅助线
- **动画速度调节** - 10级速度控制，适应不同演示需求

### 4. 教学与实验功能
- **实验记录** - 每次算法执行自动生成实验记录
- **步骤回放** - 支持实验记录的步骤级回放
- **性能对比** - 不同算法在同一场景下的性能对比
- **统计信息** - 实时显示算法复杂度、执行时间等指标

### 5. 用户管理
- **多角色系统** - 管理员、教师、学生不同权限
- **实验历史** - 用户个人的实验记录查看
- **统计分析** - 用户实验数据的统计和分析
- **学习进度** - 算法掌握程度和学习轨迹

### 1. API函数定义（src/api/user.js）

```javascript
// 用户注册
export function register(data) {
  return request({
    url: '/api/user/register',
    method: 'post',
    data
  })
}
```

### 2. 请求参数（Request Body）

| 字段名 | 类型 | 必填 | 说明 | 示例 | |--------|------|------|------|------| | username | string | 是 | 用户名，长度不少于3位 | "student123" | | password | string | 是 | 密码，长度不少于6位 | "password123" | | studentId | string | 是 | 学号 | "20210001" | | name | string | 否 | 姓名（可选） | "张三" | | email | string | 否 | 邮箱（可选） | "zhangsan@example.com" |

### 3. 请求示例

```javascript
// 前端调用示例
import { register } from '@/api/user'

const registerData = {
  username: 'student123',
  password: 'password123',
  studentId: '20210002',
  name: '李四',
  email: 'lisi@example.com'
}

// 调用注册接口
const response = await register(registerData)
```
