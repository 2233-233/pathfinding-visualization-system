# 前端导出功能实现指南

## 概述
后端已经实现了实验记录的Excel导出功能。前端需要添加导出按钮，调用相应的API接口来下载Excel文件。

## 后端API接口

### 实验记录导出接口
- **URL**: `GET /api/experiments/export`
- **参数**:
  - `userId` (可选): 用户ID，管理员可以指定，普通用户只能导出自己的
  - `algorithmId` (可选): 算法ID筛选
  - `status` (可选): 实验状态筛选
  - `startTimeFrom` (可选): 开始时间起始筛选 (格式: yyyy-MM-dd)
  - `endTimeTo` (可选): 结束时间结束筛选 (格式: yyyy-MM-dd)
- **权限**: 需要登录，管理员可以导出所有用户数据，普通用户只能导出自己的数据
- **响应**: 直接返回Excel文件流

## 前端实现步骤

### 步骤1: 创建导出工具函数

```javascript
// utils/exportUtils.js
import axios from 'axios';

/**
 * 通用导出函数
 * @param {string} url - 导出API地址
 * @param {object} params - 查询参数
 * @param {string} filename - 默认文件名
 */
export const exportToExcel = async (url, params = {}, filename = 'export.xlsx') => {
  try {
    // 获取token
    const token = localStorage.getItem('token') || sessionStorage.getItem('token');
    
    // 构建查询字符串
    const queryString = Object.keys(params)
      .filter(key => params[key] !== undefined && params[key] !== null && params[key] !== '')
      .map(key => `${encodeURIComponent(key)}=${encodeURIComponent(params[key])}`)
      .join('&');
    
    const fullUrl = queryString ? `${url}?${queryString}` : url;
    
    // 使用axios下载文件
    const response = await axios({
      url: fullUrl,
      method: 'GET',
      responseType: 'blob',
      headers: {
        'Authorization': `Bearer ${token}`,
        'Content-Type': 'application/json'
      }
    });
    
    // 创建下载链接
    const blob = new Blob([response.data], { 
      type: 'application/vnd.openxmlformats-officedocument.spreadsheetml.sheet' 
    });
    const downloadUrl = window.URL.createObjectURL(blob);
    const link = document.createElement('a');
    link.href = downloadUrl;
    link.download = filename;
    document.body.appendChild(link);
    link.click();
    document.body.removeChild(link);
    window.URL.revokeObjectURL(downloadUrl);
    
    return { success: true };
  } catch (error) {
    console.error('导出失败:', error);
    
    // 尝试读取错误信息
    if (error.response && error.response.data) {
      const reader = new FileReader();
      reader.onload = () => {
        console.error('导出错误详情:', reader.result);
      };
      reader.readAsText(error.response.data);
    }
    
    return { 
      success: false, 
      error: error.message || '导出失败，请检查网络连接或权限' 
    };
  }
};

/**
 * 导出实验记录
 * @param {object} filters - 筛选条件
 */
export const exportExperimentRecords = async (filters = {}) => {
  const filename = `实验记录_${new Date().toISOString().slice(0, 19).replace(/:/g, '-')}.xlsx`;
  return await exportToExcel('/api/experiments/export', filters, filename);
};
```

### 步骤2: 在实验记录页面添加导出按钮

```javascript
// 在实验记录列表页面添加导出按钮
import { exportExperimentRecords } from '@/utils/exportUtils';

// 在组件中添加导出函数
const handleExport = async () => {
  // 获取当前筛选条件
  const filters = {
    userId: currentUserId, // 当前用户ID，如果是管理员可以设置为null
    algorithmId: searchParams.algorithmId,
    status: searchParams.status,
    startTimeFrom: searchParams.startTimeFrom,
    endTimeTo: searchParams.endTimeTo
  };
  
  // 显示加载状态
  setExporting(true);
  
  const result = await exportExperimentRecords(filters);
  
  setExporting(false);
  
  if (!result.success) {
    message.error(result.error || '导出失败');
  } else {
    message.success('导出成功');
  }
};

// 在页面中添加导出按钮
<Button 
  type="primary" 
  icon={<DownloadOutlined />}
  onClick={handleExport}
  loading={exporting}
>
  导出Excel
</Button>
```

// 导出筛选表单组件
const ExportFilterForm = ({ onExport, initialFilters = {} }) => {

## 注意事项

1. **权限控制**: 普通用户只能导出自己的数据，管理员可以导出所有用户数据
2. **文件命名**: 后端会自动生成包含时间戳的文件名，前端也可以自定义
3. **错误处理**: 需要处理网络错误、权限错误等异常情况
4. **加载状态**: 导出过程中需要显示加载状态，避免用户重复点击
5. **筛选条件**: 导出时可以使用当前页面的筛选条件，确保导出的数据与显示的数据一致
