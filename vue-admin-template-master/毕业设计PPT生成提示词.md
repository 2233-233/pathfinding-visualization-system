# 毕业设计PPT生成提示词 - 寻路算法可视化教学平台

## 一、项目概述
**项目名称**：基于Spring Boot+Vue的寻路算法可视化系统的设计与实现
**项目类型**：软件工程专业毕业设计  
**核心目标**：开发交互式算法教学平台，通过可视化演示帮助学生理解寻路算法  
**解决痛点**：传统算法教学抽象难懂，缺乏直观演示和实践环节  

## 二、技术架构

### 前端技术栈
- **框架**：Vue.js 3 + Vue Router + Vuex
- **UI组件**：Element Plus
- **样式**：SCSS
- **HTTP客户端**：Axios
- **构建工具**：Webpack

### 后端技术栈  
- **框架**：Spring Boot 2.7.18
- **ORM**：MyBatis-Plus
- **数据库**：MySQL 8.0
- **安全认证**：JWT
- **API文档**：SpringDoc + Swagger UI

### 架构特点
- **前后端分离**：RESTful API设计
- **模块化设计**：算法、可视化、控制、数据模块
- **响应式布局**：支持多设备
- **安全认证**：RBAC权限控制

## 三、核心功能演示

### 1. 算法可视化
- **支持算法**：BFS、DFS、Dijkstra、A*（可扩展）
- **实时动画**：逐帧展示，10级速度调节
- **交互控制**：暂停、继续、重置
- **地图编辑**：设置起点、终点、障碍物

### 2. LeetCode集成
- **题库对接**：LeetCode官方题库
- **学习跟踪**：完成状态、尝试次数、学习时间
- **进度管理**：未开始/进行中/已完成
- **学习笔记**：解题思路记录

### 3. 实验数据管理
- **实验记录**：算法执行自动记录
- **数据统计**：路径长度、访问节点数、执行时间
- **历史回放**：步骤级重放功能
- **对比分析**：多算法性能对比

### 4. 用户管理
- **多角色系统**：管理员、教师、学生
- **学习统计**：个人进度和效果评估
- **实验历史**：查看和对比记录

## 四、技术实现亮点

### 1. 算法可视化技术
- **面向对象设计**：算法基类+子类继承
- **实时动画渲染**：批量更新优化性能
- **性能优化**：网格自适应，延迟计算

### 2. 前后端交互
- **RESTful API**：50+标准化接口
- **统一响应格式**：R<T>对象
- **JWT认证**：Token生成、验证、刷新
- **分页查询**：多条件筛选

### 3. 数据持久化
- **逻辑删除**：is_deleted字段
- **关系设计**：多表关联
- **步骤存储**：算法执行详细记录
